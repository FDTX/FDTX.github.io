题目—图书爬虫 模拟登陆

URL：http://222.24.3.7:8080/opac_two/reader/infoList.jsp
将借阅信息操作处理,不需要使用seleinum

爬虫实现模拟登陆的几种方法

  1. 添加 Cookie 方法：先登录将获取到的 Cookie 加入 Headers 中，最后用 GET 方法请求登录；

cookie是指某些网站为了辨别用户身份、进行 session 跟踪而储存在用户本地终端上的数据（通常经过加密）

  1. POST 请求方法：需要在后台获取登录的 URL并填写请求体参数，然后 POST 请求登录；
  2. Selenium 模拟登录：代替手工操作，自动完成账号和密码的输入。

1.进行模拟登陆

目标登录界面如下：

需要先登录才能看到图书借阅等信息，不用输入验证码。下面我用自己的账号和密码。来实现模拟登陆。

获取Cookie

我们要先登录，获取Cookie，然后把Cookies添加到Headers中去，用GET方法请求即可。
头信息如下：

模拟登陆

import requests                #http库
url = 'http://222.24.3.7:8080/opac_two/reader/infoList.jsp'
headers = {
'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:66.0) Gecko/20100101 Firefox/66.0',
'Cookie': 'JSESSIONID=282BFAA4C6B2BF2B6D1EBE08C75B60EC'}
response = requests.get(url)   #模拟服务器发送get请求
response.encoding = 'gbk'      #编码方式
html = response.text           #目标主页网页源码、
print(response.status_code)    #获取响应状态码，返回200即成功登录
print(html)

可以看到响应码是200，成功登录并获取了网页内容：

2.获取借阅历史超链接

分析爬取到的源码，找到借阅历史

如下图：

由于网页中id是唯一的，所以用正则表达式定位借阅历史的超链接，代码如下：

div = re.findall(r'<div id="pageLinks" >.*?</div>',html,re.S)[0]   #不匹配不可见字符，加re.S匹配所有字符 ,[0]从列表中取出元素
jyls = re.findall(r'href="(.*?)"',div)[6]                          #jyls--借阅历史，拿到它的a标签

jylss= "http://222.24.3.7:8080/opac_two/reader/%s" % jyls          #补全超链接

返回借阅历史的超链接

3.把借阅历史提取出来

代码如下

response = requests.get(jylss,headers=headers)                     #模拟服务器发送get请求
response.encoding = 'gbk'                                          #编码方式
jylss_html = response.text
jy = re.findall(r'<table align=left>(.*?)</table>',jylss_html,re.S)[0]
jyxq = re.findall(r'<td align=left >(.*?)</td>',jy,re.S)           #借阅详情
print(jyxq)

4.清洗，处理数据

可以看到数据里有一些我们不需要的字符，如” ”,而且数据比较凌乱

for i in range(len(jyxq)):                  
    jyxq[i] = jyxq[i].replace('&nbsp;','')         #用空字符循环替换&nbsp;

i = 0
while i<len(jyxq):                                 #五个一组循环输出
    print(jyxq[i:i+5])
    i += 5

5.数据持久化

把数据保存在文件里使数据持久化

path = "D:/图书借阅信息.txt"          #在D盘创建图书借阅信息文本
with open(path,'w') as f:           #打开文件
    i = 0                           
    while i < len(jyxq):            
        f.write(str(jyxq[i:i+5]))   
        f.write("\n")
        i += 5
    f.close()                       #关闭文件
